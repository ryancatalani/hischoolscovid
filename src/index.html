Hawaiʻi Public Schools: Reported COVID-19 Cases

Compiled by Ryan Catalani (@ryancatalani)

About the data & submit missing data

Data updated daily on weekdays.

This data is primarily compiled from the Department of Education data dashboard, starting from July 24, 2021. Public charter schools appear not to be included in the data.

If you have any data that’s missing, or see something inaccurate, click here to submit additional data. Public data submissions are indicated in the list below.

School enrollment figures are from the 2021-22 school year, via the DOE. The latest available data on the number of teachers and administrative staff is from the 2019-20 school year, via the DOE. Administrative staff figures are only available as FTEs per school, rather than headcount.
